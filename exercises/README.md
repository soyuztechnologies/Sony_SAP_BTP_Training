# Exercises

This directory contains hands-on exercises to reinforce your learning from each module.

## Structure

Each exercise is organized in its own directory and includes:
- **README.md**: Exercise instructions and objectives
- **starter-code/**: Initial code to begin the exercise
- **solution/**: Complete solution for reference
- **assets/**: Any additional files needed

## Available Exercises

### Module 1: Introduction to SAP BTP
- Exercise 1.1: Setting up your trial account
- Exercise 1.2: Exploring the BTP landscape

### Module 2: SAP BTP Cockpit
- Exercise 2.1: Creating and configuring subaccounts
- Exercise 2.2: Managing users and roles

### Module 3: Cloud Foundry Environment
- Exercise 3.1: Deploying your first application
- Exercise 3.2: Service binding and configuration

### Module 4: SAP HANA Cloud
- Exercise 4.1: Creating a HANA Cloud instance
- Exercise 4.2: Building a data model

### Module 5: SAP Business Application Studio
- Exercise 5.1: Creating a CAP project
- Exercise 5.2: Building a Fiori application

### Module 6: Integration Suite
- Exercise 6.1: Creating an API proxy
- Exercise 6.2: Building an integration flow

## How to Use Exercises

1. Read the exercise README.md carefully
2. Start with the provided starter code
3. Follow the step-by-step instructions
4. Test your implementation
5. Compare with the solution if needed

## Tips

- Don't peek at the solution too early - try to solve it yourself first
- Experiment and make mistakes - that's how you learn!
- Take notes on challenges you face
- Share your learnings with others

## Need Help?

If you're stuck on an exercise:
1. Review the relevant module content
2. Check the FAQ
3. Search the SAP Community
4. Ask for help in the discussions section
