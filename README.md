## About

This training program is designed to help developers and IT professionals learn SAP Business Technology Platform (BTP). SAP BTP is an integrated offering comprised of four technology portfolios: database and data management, application development and integration, analytics, and intelligent technologies.

## Prerequisites

Before starting this training, you should have:

- Basic understanding of cloud computing concepts
- Familiarity with web development (HTML, CSS, JavaScript)
- SAP BTP trial account or access to an SAP BTP environment
- A modern web browser (Chrome, Firefox, or Edge)
- Node.js (v14 or higher) installed on your machine
- Git installed for version control

## Getting Started

1. **Clone this repository:**
   ```bash
   git clone https://github.com/soyuztechnologies/Sony_SAP_BTP_Training.git
   cd Sony_SAP_BTP_Training
   ```

2. **Set up your SAP BTP account:**
   - Sign up for a free trial account at [SAP BTP Trial](https://www.sap.com/products/technology-platform/trial.html)
   - Access the SAP BTP Cockpit
   - Create a subaccount if not already created

3. **Explore the training modules:**
   - Start with Module 1: Introduction to SAP BTP
   - Follow the sequential order for best learning experience

## Training Modules

### Module 1: Introduction to SAP BTP
- Overview of SAP BTP architecture
- Key services and capabilities
- Account structure and management

### Module 2: SAP BTP Cockpit
- Navigating the BTP Cockpit
- Managing subaccounts and spaces
- User and role management

### Module 3: Cloud Foundry Environment
- Understanding Cloud Foundry
- Deploying applications
- Service binding and configuration

### Module 4: SAP HANA Cloud
- Introduction to SAP HANA Cloud
- Creating database instances
- Basic SQL operations

### Module 5: SAP Business Application Studio
- Setting up the development environment
- Creating CAP applications
- Developing Fiori applications

### Module 6: Integration Suite
- API Management
- Integration flows
- Connectivity options
