# Module 2: SAP BTP Cockpit

## Overview

This module covers the SAP BTP Cockpit, the central administration and management tool for your SAP BTP environment.

## Learning Objectives

By the end of this module, you will be able to:
- Navigate the SAP BTP Cockpit effectively
- Manage subaccounts and spaces
- Configure user access and authorizations
- Monitor usage and quotas

## Topics Covered

### 2.1 Cockpit Navigation
- Understanding the cockpit interface
- Global account vs. subaccount views
- Quick access and favorites

### 2.2 Subaccount Management
- Creating and configuring subaccounts
- Managing entitlements
- Setting up environments

### 2.3 User and Role Management
- Adding users to your account
- Assigning roles and role collections
- Security best practices

### 2.4 Service Marketplace
- Browsing available services
- Subscribing to applications
- Creating service instances

## Hands-on Exercise

1. Create a new subaccount
2. Assign entitlements to the subaccount
3. Add users and assign appropriate roles
4. Explore the Service Marketplace
