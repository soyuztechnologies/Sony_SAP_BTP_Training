### üìä Cheat Sheets

#### Cloud Foundry CLI Commands
```bash
# Login
cf login -a <api-endpoint>

# View apps
cf apps

# View services
cf services

# Push application
cf push <app-name>

# View logs
cf logs <app-name>

# Scale application
cf scale <app-name> -i <instances> -m <memory>

# Bind service
cf bind-service <app-name> <service-instance>

# Restage application
cf restage <app-name>
```
